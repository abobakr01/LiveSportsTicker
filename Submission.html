Project summary

This project implements Project #1: a live sports ticker using Server-Sent Events (SSE). A background service advances a simulated match clock and produces events (kick-off, goals, cards, substitutions, half-time, full-time, commentary). Events are stored in-memory and streamed to the browser over a single text/event-stream response. A live win-probability is recomputed after every event and shown in the UI.

How to run locally

  1. Open a terminal in the project root.
  2. dotnet restore
  3. dotnet run
  4. Open the URL printed by the app (e.g. http://localhost:5080) in your browser.

Files of interest

  • Controllers/HomeController.cs — serves the Razor page
  • Controllers/StreamController.cs — GET /Stream/Live, the SSE endpoint
  • Services/MatchSimulator.cs — in-memory event log + win-probability model
  • Services/MatchClockService.cs — background service driving the clock
  • Views/Home/Index.cshtml — live UI (EventSource, scoreboard, probability bar, feed)
